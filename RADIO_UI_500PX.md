# Radio UI - Идеальное заполнение 500px

## Цель
Изменить размеры элементов чтобы оба блока (левая панель плеера и правая панель плейлиста) идеально заполняли 500px по высоте.

## Расчеты

### Правая панель (Playlist)
- Высота контейнера: 500px
- Количество песен: 5 (2 предыдущих + текущая + 2 следующих)
- Высота на элемент: **500px / 5 = 100px**

### Левая панель (Player)
- Фиксированная высота: 500px
- Фиксированная ширина: 355px
- Элементы внутри увеличены пропорционально

## Изменения

### 1. Правая панель (Playlist) - элементы 100px

#### Размеры контейнера элемента
- Margins: `12, 8, 12, 8` → `16, 12, 16, 12`
- Spacing: `12` → `16`
- **Высота**: `60px` → **100px**

#### Порядковый номер
- Width: `35px` → `40px`
- Font size: `16px` → `20px`

#### Обложка альбома
- Размер: `50x50` → **70x70px**

#### Song info layout
- Spacing: `3px` → `6px`

#### Текст
- **Label** (PREVIOUS/NOW PLAYING/NEXT):
  - Font size: `11px` → `12px`

- **Title**:
  - Font size: `15px` → **18px**
  - Max width: `600px` → `700px`

- **Artist**:
  - Font size: `13px` → **15px**
  - Max width: `600px` → `700px`

- **Duration**:
  - Width: `50px` → `60px`
  - Font size: `14px` → **16px**

**Итого**: 5 элементов × 100px = **500px** ✅

### 2. Левая панель (Player) - заполнение 500px

#### Контейнер
- **Фиксированный размер**: `setFixedSize(355, 500)`
- Margins: `10, 5, 10, 5` → `20, 15, 20, 15`
- Spacing: `3` → `10`

#### Album Art
- Размер: `80x80` → **200x200px**

#### Текст
- Artist: `8px` → `13px`
- Song Title: `10px` → `16px`
- Time labels: `7px` → `11px`

#### Прогресс-бар
- Spacing: `3` → `8`
- Groove height: `3px` → `5px`
- Handle: `8x8` → `12x12px`
- Margin: `-3px` → `-4px`
- Border radius: `1px` → `2px`

#### Кнопки управления
- Play/Pause: `24x24` → **40x40px**
- Border radius: `12px` → `20px`
- Volume button: `18x18` → **30x30px**
- Volume button font: `10px` → `16px`
- Volume slider width: `50px` → **100px**
- Control spacing: `4px` → `10px`
- Between spacing: `4px` → `10px`

#### Кнопка REQUEST SONG
- Border radius: `4px` → `6px`
- Padding: `4px 8px` → `10px 20px`
- Font size: `8px` → `12px`
